************************************User Manual*********************************************

Step1 : Enter your text to be translated in "Enter the Text to translate" box.

Step2 : Click on "Click To see language code" button and check for the code of the language 
        you want to translate in.

Step3 : Copy the code and paste it in the "Destination Language Code" box.

Step4 : Click on "Translate" to get your text translated.

# Your translation will pop on the screen.


#################### File Tab ###########################

Reset : Click to reset the inputed values and the result.

Save Results : Click to save the "INITIAL TEXT" , "SOURCE CODE OF LANGUAGE" , "DESTINATION CODE OF THE 
	       LANGUAGE " AND "TRANSLATED TEXT" to a csv(Comma Separated File).
	
Exit : Exits the program.
